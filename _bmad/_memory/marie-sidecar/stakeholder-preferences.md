## Communication Patterns

### By Audience Type

**Executives (C-Suite):**
- 2-page max, visual-first
- Business impact over technical details
- Clear recommendations with cost/benefit
- Avoid jargon, use analogies

**Technical Teams:**
- Full methodology, code included
- Statistical rigor expected
- Reproducibility critical
- Deep dive into edge cases OK

**Product Managers:**
- Feature impact focus
- User behavior insights
- A/B test results
- Prioritization guidance
